🚀 PrepEdge - AI-Powered Interview Preparation Platform
PrepEdge is a comprehensive full-stack web application designed to help students and professionals prepare for technical interviews through structured DSA practice, mock interviews, resume analysis, and personalized roadmaps.

✨ Features
Authentication System - Secure JWT-based authentication with bcrypt password hashing
Dashboard - Real-time progress tracking and analytics visualization
DSA Tracker - Topic-wise problem solving with LeetCode integration
Mock Interviews - Simulated interviews with AI-powered scoring and feedback
Resume Analysis - ATS score calculation and optimization suggestions
Skill Gap Analysis - Identify weak areas and get personalized recommendations
Smart Roadmap - Dynamic weekly preparation plans based on your progress
Analytics Dashboard - Visual insights into your preparation journey
🛠️ Tech Stack
Frontend
HTML5, CSS3, Vanilla JavaScript
Canvas API for charts and visualizations
Responsive design with CSS Grid/Flexbox
Backend
Node.js with Express.js
MongoDB Atlas for database
JWT for authentication
bcrypt for password security
Multer for file uploads
📋 Prerequisites
Node.js (v14 or higher)
MongoDB Atlas account (free tier works)
Git
